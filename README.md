
### 简介 

这是一个基于RuoYi-Vue快速开发平台二次开发的项目。
    
本项目计划使用MyBatis-Flex重构源项目中所有数据库访问部分，其他部分尽可能保持原样，本项目持续进行中，最终计划消灭源项目中所有可被flex替代的SQL(暂不包含代码生成器生成的代码)
